# Security Policy

## Reporting

Please do not open a public issue for suspected security problems.

Send a private report with:

- affected version or commit
- reproduction steps
- impact assessment
- any proposed mitigation

## Scope

This project currently covers the CLI source, CI workflow, and published repository contents.

## Response

Best effort response targets:

- initial acknowledgment within 7 days
- triage and severity assessment after reproduction
- fix and coordinated disclosure when the issue is confirmed
